# Udemy Clone
A creative Udemy-inspired webpage built using HTML and CSS with my own design ideas and responsive layout.


## Features
- Responsive layout using Flexbox and Grid  
- Custom Udemy navigation bar  
- Search bar and menu section  
- Recommended courses section  
- Recommended topics input section  
- Popular courses with ratings and reviews  
- Footer with links and copyright details  



## Technologies Used
- HTML5  
- CSS  


## Project Sections
- Navbar  
- Menu Bar  
- Recommended Courses  
- Pick right Topics  
- Popular Courses  
- Footer  



## Purpose
This project was created to practice frontend web development and improve HTML/CSS layout skills by recreating the Udemy homepage design.
